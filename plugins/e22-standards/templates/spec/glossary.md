# Glossary

Shared vocabulary for this product. Add a term whenever PO, devs, and Claude would otherwise use different words for the same thing — or the same word for different things.

## How to use this file

- If you find yourself re-explaining what a term means in a spec, issue, or PR comment, add it here.
- Claude reads this file. Consistent vocabulary here means consistent vocabulary in generated specs and code.
- Keep entries to one or two sentences. This is a quick reference, not a dictionary.
- When you rename something in code, update this file in the same PR.
